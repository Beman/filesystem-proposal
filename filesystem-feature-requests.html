Filesystem Feature Requests

The C++ Standards Committee's Filesystem Library Technical Specification 1 (Filesystem TS1) is closed for new features. This informal Study Group 3 document tracks new feature requests so that they will not be lost and can be reviewed when work starts on a new TS or a revision of TS1.
