copy_options

Options controlling copy_file effects when target file already exists
Constant Value Meaning
none 0 (Default) Error; file already exists.
skip 1 Do not overwrite existing file, do not report an error.
overwrite 2 Overwrite the existing file.
update 3 Overwrite the existing file if it is older than the replacement file.

 

Options controlling copy effects for sub-directories
Constant Value Meaning
none 0 (Default) Do not copy sub-directories.
recursive 0x04 Recursively copy sub-directories and their contents.
